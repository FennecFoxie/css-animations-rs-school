Keyframes

Style can be changed by using the keywords "from" and "to" (which represent 0% (start) and 100% (complete)). It is also possible to use percent. By using percent, you can add as many style changes as you like. 

            @keyframes example {
              from {background-color: red;}
              to {background-color: yellow;}
            }
        
            @keyframes example {
              0% {background-color: red;}
              5% {background-color: yellow;}
              50% {background-color: blue;}
              100% {background-color: green;}
            }

Animation

  • animation-name
  • animation-duration
  • animation-timing-function
  • animation-delay
  • animation-iteration-count (number or infinite)
  • animation-direction (normal, reverse, alternate, alternate-reverse)
  • animation-play-state (paused or running)
  • animation-fill-mode (none, forwards, backwards, both)
            @keyframes preloader {
              0% { transform: scale(1); }
              100% { transform: scale(2); }
            }
        
            animation: preloader .6s ease-in-out alternate .4s infinite;
